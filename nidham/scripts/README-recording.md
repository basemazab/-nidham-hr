# تسجيل فيديوهات شرح النظام

## المتطلبات
1. npm install
2. npx playwright install chromium
3. عبّي `.env.local` بالبيانات المطلوبة

## التشغيل
```bash
npx tsx scripts/record-video-walkthrough.ts
```

الفيديوهات بتظهر في:
`~/Desktop/نظام_HR_يوتيوب/فيديوهات_شرح/`
